## M_T_Gaming Store Website

A modern gaming store website built using HTML, CSS, and JavaScript. Features include an animated hero section, interactive game cards, signup popup authentication using Local Storage, smooth UI animations, responsive design, and a premium streaming-platform-inspired user experience.

 Features:
• Animated Hero Banner
• Interactive Game Cards
• Signup/Login Popup
• Local Storage Authentication
• Search Functionality
• Smooth Animations & Hover Effects
• Responsive Design
• Modern Dark Theme UI

 Built with:
HTML5 | CSS3 | JavaScript
